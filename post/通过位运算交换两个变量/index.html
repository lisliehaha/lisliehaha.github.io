通过位运算交换两个变量

2019-07-27
Java
约 173 字 预计阅读 1 分钟 次阅读

文章目录

有两个变量 a 和 b ，想交换这两个变量的值；

1.常规方式

思路：定义一个临时变量储存其中一个值，然后进行交换。

String temp=a; a=b; b=temp;

2.位运算方式

原理： 一个变量按位异或自己结果为0：a^a=0 一个变量按位异或0结果为自己：a^0=a 异或运算的逻辑符号图： 通过逻辑图可以很容易理解上面两个基本逻辑式

则a ，b变量交换可以通过异或计算的方式实现：

1
2
3

>     a^=b ;    //a=a^b
>     b^=a;     //b=b^a=b^a^b=0^a=a
>     a^=b;    //a=a^b=a^b^a=b
