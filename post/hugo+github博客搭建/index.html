Hugo+Github博客搭建

2019-07-20
软件安装
约 231 字 预计阅读 1 分钟 次阅读

文章目录

1.hugo安装

参见官网：hugo官网 注：需要配置环境变量

2.新建站点

$ hugo new site myBlog

3.下载hugo主题

将主题克隆到themes文件夹中

$ git clone https://github.com/olOwOlo/hugo-theme-even themes/even

4.新建博客&本地调试

在站点根目录执行：

$ hugo new post/firstblog.md

启动本地调试：

hugo server -t even -D

5.部署到Github

新建一个repo，命名为 yourname.github.io ,执行：

hugo –theme=even –baseUrl=“https://youname.github.io"

之后目录下生成一个public文件夹，进入里面

git add -A

git commint -m “create blog”

git remote add origin https://github.com/youname/youname.github.io.git

git push -u origin master

按提示输入git用户，密码
